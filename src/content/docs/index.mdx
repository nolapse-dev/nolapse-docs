## What is Nolapse?

Nolapse is a git-native coverage enforcement CLI and GitHub Action. It computes the
coverage delta between a PR branch and the baseline, applies your configured thresholds,
and fails the PR when coverage regresses below your policy.

**Supported languages:** Go · Python · Node.js · Java · .NET

**North star:** A developer can go from zero to their first successful `nolapse run`
in under 30 minutes using only these docs.
